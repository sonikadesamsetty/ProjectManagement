Project Management Tool

Plan sprints, manage stories, track tasks — collaborate with clarity.

What is this app?

This is a lightweight Project Management Tool built to help teams organise work into projects, sprints and cards. Create stories, add tasks, move cards across columns and monitor sprint progress.

Boards

Kanban-style boards to visualise work and flow items from To Do → In Progress → Done.

Sprints

Plan timeboxed sprints and track completion of stories and tasks during each sprint.

Collaboration

Assign users, add comments and maintain a clear history of changes per item.

Quick Start

  1. Create a project.
  2. Add stories and break them into tasks.
  3. Start a sprint and move tasks across the board.
